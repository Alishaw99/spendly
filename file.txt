❯ what is jinja templating engine                                               

● Jinja is a templating engine for Python. It lets you embed dynamic content
  into HTML files using special syntax.

  Key syntax:

  ┌────────┬────────────────────────┬─────────────────────────┐
  │ Syntax │        Purpose         │         Example         │
  ├────────┼────────────────────────┼─────────────────────────┤
  │ {{ }}  │ Output a variable      │ {{ user.name }}         │
  ├────────┼────────────────────────┼─────────────────────────┤
  │ {% %}  │ Logic (if, for, block) │ {% if logged_in %}      │
  ├────────┼────────────────────────┼─────────────────────────┤
  │ {# #}  │ Comments               │ {# this is a comment #} │
  └────────┴────────────────────────┴─────────────────────────┘

  Example:
  <!-- templates/index.html -->
  <h1>Hello, {{ name }}!</h1>

  {% for expense in expenses %}
    <p>{{ expense.title }}: ${{ expense.amount }}</p>
  {% endfor %}

  {% if expenses %}
    <p>Total: {{ total }}</p>
  {% else %}
    <p>No expenses yet.</p>
  {% endif %}

  In Flask, it's built-in — you just call render_template() and pass variables:
  return render_template("index.html", name="Tariq", expenses=expenses,
  total=100)

  Flask automatically looks for templates in the templates/ folder and renders
  them with Jinja.

  It's essentially HTML with Python-like logic baked in — very relevant for your
   Spendly app.
